Emmet

. - Dot after the element determines the class
> - More than means the following element will be an html child
+ - Plus means the following element will be an adjacent html element

Example:
  Shortcut:
    div.form-element>label.form-label+input.form-input

  Output:
    <label htmlFor="" className="form-label"></label>
    <input type="text" className="form-input" />
