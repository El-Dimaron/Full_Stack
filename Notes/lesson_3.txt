Абсолютні одиниці вимірювання в CSS використовуються для вказівки точних розмірів елементів, які не залежать від інших елементів чи налаштувань користувача.
До них відносяться пікселі (px), пункти (pt), дюйми (in), сантиметри (cm) та міліметри (mm).

Відносні одиниці вимірювання (основні):
1. % (відсоток): Значення відносно розміру батьківського елемента.
2. em: Значення відносно розміру шрифту елемента.
3. ex: Відноситься до висоти маленької літери x поточного шрифту.
4. ch: Відноситься до ширини символу 0 (нуль) поточного шрифту.
5. rem: Відносно розміру шрифту кореневого елемента (html).
6. vw: Відсоток від ширини вікна перегляду.
7. vh: Відсоток від висоти вікна перегляду.
8. vmin: Відсоток від меншої сторони вікна перегляду.
9. vmax: Відсоток від більшої сторони вікна перегляду.




Властивості font:
  font-family: визначає сімейство шрифтів для тексту.
  font-style: встановлює стиль шрифту, наприклад, нормальний, курсив або нахилений.
  font-variant: дозволяє використовувати варіації шрифту, наприклад, маленькі великі літери.
  font-weight: встановлює товщину шрифту, наприклад, нормальна, жирна, або конкретне числове значення.
  font-size: визначає розмір шрифту.


Властивості text-align, text-decoration, та text-transform у CSS використовуються для стилізації тексту на веб-сторінках:
  text-align
    визначає горизонтальне вирівнювання тексту усередині елемента. Можливі значення: left, right, center, justify.
  text-decoration
    використовується для задання оформлення тексту, наприклад, підкреслення, перекреслення, або верхньої лінії. Можливі значення: underline, overline, line-through, none.
  text-transform
    використовується для зміни регістру тексту. Можливі значення: uppercase, lowercase, capitalize, none.


Властивості list-style, list-style-position, list-style-image, та list-style-type у CSS використовуються для стилізації маркованих та нумерованих списків.

  list-style-type
    визначає тип маркера для пунктів списку. Наприклад, disc, circle, square для маркованих списків та decimal, lower-alpha, upper-roman для нумерованих списків.
  list-style-position
    встановлює положення маркера відносно пунктів списку. Можливі значення: inside, outside.
  list-style-image
    дозволяє використовувати зображення в якості маркера пунктів списку.
  list-style
    є складовою властивістю, що дозволяє одночасно задати list-style-type, list-style-position та list-style-image.
