CSS Grid - це модуль в CSS, який дає можливість будувати сітчасті макети та розміщувати елементи сторінки у двовимірній системі рядків і стовпців.

Модуль вирівнювання (alignment) в CSS Grid та Flexbox має багато спільного.

Обидві технології використовують схожі за функціоналом властивості для вирівнювання елементів:
  justify-content та align-content для вирівнювання контейнера
  justify-items та align-items для вирівнювання всіх елементів в контейнері
  justify-self та align-self для вирівнювання окремих елементів

Також присутні скорочені властивості place-content, place-items та place-self



fr units

  fr - це одиниця вимірювання в CSS Grid, яка репрезентує частку вільного простору в межах контейнера сітки.

  На відміну від px чи %, fr є гнучкою одиницею і динамічно підлаштовує розмір треку сітки в залежності від загальної ширини чи висоти контейнера.

display: grid | inline-grid;
  grid: позначає елемент, як контейнер сітки (grid container).
  Це означає, що всі його дочірні елементи стануть елементами сітки (grid items) і їх можна буде розміщувати за допомогою властивостей grid.

  inline-grid: робить те саме, що і grid, але контейнер сітки стає inline-елементом.


grid-template-columns grid-template-rows
  Визначають стовпці та рядки сітки.

  Ці властивості задають ширину кожного стовпця та висоту кожного рядка сітки CSS.
  Вони не тільки задають розміри, але й кількість стовпців та рядків. Це дозволяє створювати сітки довільної структури та розмірів.

  Приклад:
    .container {
        grid-template-columns: 40px 50px auto 50px 40px;
        grid-template-rows: 25% 100px auto;
    }


grid-template-areas
  Дозволяє визначати розмітку сітки за допомогою імен областей (grid areas).

  За допомогою grid-template-areas ми можемо вказати як саме мають розміщуватися елементи сітки на сторінці, не задаючи конкретні рядки чи стовпці, а лише посилаючись на імена областей.

grid-area
  Дає можливість призначити ім'я для елемента сітки, щоб посилатися на нього при використанні grid-template-areas.

  Крім того, grid-area можна використовувати як ще більш короткий запис для визначення розташування елемента в сітці за допомогою початкового рядка/стовпця та кінцевого рядка/стовпця.

  Шаблон:
    grid-area: <name> | <row-start> / <column-start> / <row-end> / <column-end>;


grid-template
  Скорочена властивість, яка дозволяє задати grid-template-rows, grid-template-columns та grid-template-areas в одному рядку.

    .container {
      grid-template: none | <grid-template-rows> / <grid-template-columns>;
    }



grid-column-start grid-column-end grid-row-start grid-row-end
  Визначають розташування конкретного елемента сітки на сторінці.

  За допомогою цих властивостей, можна точно задати комірку сітки, у якій повинен розміщуватися певний елемент. grid-column-start і grid-row-start вказують початкові лінії, а grid-column-end і grid-row-end - кінцеві.

  Шаблон:
    .item {
      grid-column-start: <number> | <name> | span <number> | span <name> | auto;
      grid-column-end: <number> | <name> | span <number> | span <name> | auto;
      grid-row-start: <number> | <name> | span <number> | span <name> | auto;
      grid-row-end: <number> | <name> | span <number> | span <name> | auto;
    }

  Приклад:
    .item-a {
      grid-column-start: 2;
      grid-column-end: five;
      grid-row-start: row1-start;
      grid-row-end: 3;
    }


grid-column grid-row
  Скорочені властивості, які дозволяють задати початкову та кінцеву лінії для розміщення елемента сітки в одному рядку коду.

  Шаблон:
    .item {
      grid-column: <start-line> / <end-line> | <start-line> / span <value>;
      grid-row: <start-line> / <end-line> | <start-line> / span <value>;
    }


justify-self: start | end | center | stretch;
  Горизонтально вирівнює окремий елемент сітки всередині його комірки.


align-self: start | end | center | stretch;
  Вертикально вирівнює окремий елемент сітки всередині його комірки.


place-self: <align> <justify>;
  Скорочена властивість, яка дозволяє задати значення для align-self та justify-self в одному рядку

  Або просто для центрування: place-self: center;




justify-items: start | end | center | stretch;
  Горизонтально вирівнює всі елементи сітки всередині їхніх комірок.

align-items: start | end | center | stretch;
  Вертикально вирівнює всі елементи сітки всередині їхніх комірок.

На відміну від justify-self/align-self, які впливають на окремий елемент, justify-items/align-items встановлюють вирівнювання для всіх елементів сітки відразу за один рядок коду.


place-items: <align-items> <justify-items>;
  Скорочена властивість, яка дозволяє одночасно задати значення властивостей align-items та justify-items для вирівнювання елементів сітки по горизонталі та вертикалі.

  Центрування (шорткат): place-items: center;


justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
  Вирівнює всю сітку по горизонталі відносно контейнера, у випадку, коли сітка менша за нього.

  Використовуючи justify-content, ми вказуємо, як саме grid має розміщуватися в контейнері, якщо не заповнює всю його ширину: по лівому краю, правому, центруватися, розтягуватися на всю ширину тощо.


align-content: start | end | center | stretch | space-around | space-between | space-evenly;
  Вирівнює всю сітку по вертикалі відносно контейнера, у випадку, коли сітка менша за нього.

  Використовуючи align-content, ми вказуємо, як саме grid має розміщуватися в контейнері, якщо не заповнює всю його висоту: зверху, знизу, центруватися, розтягуватися на всю висоту тощо.


place-content: <align-content> <justify-content>;
  Скорочена властивість в CSS Grid, що дозволяє встановити значення для align-content та justify-content в одному рядку коду.


column-gap row-gap
  Використовуються для створення відступів між колонками і рядками нашої сітки.

  За допомогою цих властивостей ми можемо точно контролювати ширину відступів між рядками і стовпцями сітки, не впливаючи на розміри самих треків сітки.

gap
  Скорочена властивість для одночасного встановлення відступів між рядками та стовпцями сітки



grid-auto-columns grid-auto-rows
  Використовуються для визначення розмірів автоматично генерованих колонок і рядків сітки.

  Коли ми додаємо в сітку більше елементів, ніж визначено колонок/рядків, або розміщуємо елементи за межами сітки - автоматично створюються додаткові комірки.
  Саме розміри цих комірок ми і контролюємо за допомогою grid-auto-columns та grid-auto-rows. Це дає змогу регулювати розміри автоматично генерованої частини сітки.

  Приклад:
    grid-auto-columns: 60px;


grid-auto-flow: row | column | row dense | column dense;
  Визначає алгоритм автоматичного розміщення елементів сітки, позиція яких не задана явно.

  За замовчуванням, алгоритм заповнює кожний рядок послідовно, створюючи нові рядки при потребі.

  Але за допомогою grid-auto-flow ми можемо змінити цю поведінку: заповнювати стовпці, заповнювати пусті ділянки на сітці щільніше тощо.
