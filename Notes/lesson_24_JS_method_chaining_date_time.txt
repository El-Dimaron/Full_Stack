Ланцюгові виклики методів (Method chaining)
  Ланцюгові виклики методів - це підхід, що дозволяє нам викликати кілька методів один за одним в одному рядку коду, використовуючи повернене значення попереднього методу як вхід для наступного. Це дуже зручно для зменшення кількості коду та підвищення його читабельності, особливо коли ми працюємо з масивами або об'єктами, які мають ланцюжок методів для обробки даних.

  Приклад коду:
    Уявімо, що у нас є масив чисел, і ми хочемо спочатку відфільтрувати числа, що є більшими за 5, потім відсортувати ці числа в порядку зростання, та нарешті помножити кожне число на 2. Використовуючи ланцюгові виклики методів, ми можемо зробити це все в одному рядку.
      const numbers = [1, 6, 3, 8, 4, 2, 7];

      const processedNumbers = numbers
        .filter(number => number > 5)  // Відфільтрувати числа більші за 5
        .sort((a, b) => a - b)         // Відсортувати за зростанням
        .map(number => number * 2);    // Помножити кожне число на 2

      console.log(processedNumbers);   // Вивести результат: [12, 14, 16]

Дата та Час (Date)
  Дата та час у JavaScript представлені об'єктом Date, який надає методи для роботи з датами та часом. Об'єкт Date може містити інформацію про рік, місяць, день, годину, хвилину, секунду та мілісекунду. Ми можемо створювати нові об'єкти Date, отримувати поточну дату та час, маніпулювати датами (додавати чи віднімати дні, місяці тощо), а також форматувати вивід дати та часу для відображення у зручному форматі.

  Варіанти створення дати
    В JavaScript існує кілька способів створення об'єктів Date, кожен з яких призначений для різних сценаріїв використання:
      new Date() - створює об'єкт Date, який представляє поточну дату та час.
      new Date(milliseconds) - створює об'єкт Date, представляючи дату та час, визначені в мілісекундах від початку Unix епохи (1 січня 1970 00:00:00 UTC).
      new Date(dateString) - створює об'єкт Date на основі рядкового представлення дати.
      new Date(year, month, date, hours, minutes, seconds, ms) - створює об'єкт Date з вказаними компонентами. Місяці починаються з 0 (тобто, 0 для січня).

  Отримання компонентів дати
    Для отримання компонентів дати в JavaScript, ми використовуємо методи об'єкта Date. Ці методи дозволяють отримати рік, місяць, день, годину, хвилину, секунду та мілісекунду конкретної дати. Кожен компонент дати доступний через відповідний метод, який повертає його значення.

    Приклад коду:
      Створимо об'єкт Date для поточної дати та часу, а потім отримаємо та виведемо окремі компоненти цієї дати:
        const now = new Date(); // Створюємо об'єкт для поточної дати та часу

        // Отримуємо компоненти дати
        const year = now.getFullYear(); // Рік
        const month = now.getMonth() + 1; // Місяць (додаємо 1, тому що місяці починаються з 0)
        const day = now.getDate(); // День
        const hours = now.getHours(); // Години
        const minutes = now.getMinutes(); // Хвилини
        const seconds = now.getSeconds(); // Секунди

        // Виводимо компоненти
        console.log(`Рік: ${year}, Місяць: ${month}, День: ${day}, Час: ${hours}:${minutes}:${seconds}`);

  Встановлення компонентів дати
    Для встановлення компонентів дати в JavaScript, ми використовуємо відповідні методи об'єкта Date. Ці методи дозволяють змінити рік, місяць, день, годину, хвилину, секунду або мілісекунди конкретного об'єкта Date. Кожен компонент може бути встановлений окремо, використовуючи методи, які починаються з set, наприклад, setFullYear(), setMonth(), setDate(), setHours(), setMinutes(), setSeconds(), setMilliseconds().

    Приклад коду:
    Змінимо дату та час для об'єкта Date, встановивши нові значення для року, місяця, дня та години:
      const date = new Date(); // Створюємо об'єкт для поточної дати та часу
      console.log('Початкова дата:', date);

      // Встановлюємо нові компоненти дати
      date.setFullYear(2024); // Встановлюємо рік
      date.setMonth(0); // Встановлюємо місяць (0 = січень, оскільки в JavaScript місяці починаються з 0)
      date.setDate(1); // Встановлюємо день
      date.setHours(12); // Встановлюємо годину
      date.setMinutes(0); // Встановлюємо хвилини
      date.setSeconds(0); // Встановлюємо секунди

      console.log('Змінена дата:', date);

  Пояснення корисних методів та функцій роботи з датою та часом
    1. Date.now() - повертає поточну дату та час у мілісекундах від початку Unix епохи (1 січня 1970).
    2. toISOString() - конвертує дату в рядок у форматі ISO 8601 (YYYY-MM-DDTHH:mm:ss.sssZ).
    3. toLocaleString() - конвертує дату та час у рядок, що відповідає локалі користувача.
    4. performance.now() - повертає час у мілісекундах, виміряний від початку завантаження сторінки. Використовується для точного вимірювання продуктивності.
    5. console.time - починає таймер, який можна використовувати для вимірювання тривалості операції.
    6. console.timeEnd - зупиняє таймер, початий з console.time, і виводить тривалість в мілісекундах.

    Приклади коду:
      // Date.now()
      console.log('Поточний час (мс від Unix епохи):', Date.now());

      // toISOString()
      const now = new Date();
      console.log('ISO строка поточної дати:', now.toISOString());

      // toLocaleString()
      console.log('Локалізована строка поточної дати:', now.toLocaleString('uk-UA'));

      // performance.now()
      console.log('Час виконання (мс від початку завантаження сторінки):', performance.now());

      // console.time та console.timeEnd
      console.time('Вимірювання');
      for(let i = 0; i < 1000000; i++) {} // Певна операція для вимірювання
      console.timeEnd('Вимірювання');
