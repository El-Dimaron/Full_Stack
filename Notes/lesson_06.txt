Таблиці

Мінімальна структура
Структура таблиці в HTML використовується для організації і відображення даних у формі рядків і стовпців.
  Мінімально потрібні теги для створення таблиці - це <table>, <tr> та <td>. <table> визначає контейнер таблиці, <tr> використовується для створення рядка, а <td> визначає окрему комірку в рядку.


Повна структура таблиці
Для створення більш структурованих та доступних таблиць у HTML, окрім основних тегів <table>, <tr> та <td>, можна використовувати наступні додаткові теги:

  <th>: Визначає заголовок стовпця або рядка. Заголовки зазвичай відображаються виділеним шрифтом.
  <thead>: Групує один або декілька рядків, які використовуються як заголовок таблиці.
  <tbody>: Групує основний вміст таблиці. Використання <tbody> допомагає в структуруванні даних та може поліпшити доступність.
  <tfoot>: Групує один або декілька рядків, які використовуються як нижній колонтитул таблиці. Це корисно для підсумкових рядків.
  <caption>: Визначає заголовок або опис таблиці. Зазвичай розміщується над таблицею.


Об'єднання комірок
Об'єднання комірок в таблиці HTML - це техніка, яка дозволяє об'єднувати дві або більше суміжні комірки в одну більшу комірку. Це використовується для створення складних макетів таблиць, злиття комірок по горизонталі (colspan) або по вертикалі (rowspan).


Стильове оформлення таблиць в CSS

Приклад:
.styled-table {
  border: 1px solid #000;
  width: 400px;
  height: 200px;
  border-collapse: collapse;
  border-spacing: 10px;
}

border-collapse використовується для схлопування меж комірок між собою та з межами самої таблиці.


Шрифти

Web fonts (embedded fonts)
  Web fonts (або вбудовані шрифти) - це шрифти, які ми включаємо безпосередньо в наші веб-проекти.
  Ми використовуємо вбудовані шрифти, щоб забезпечити однакове відображення тексту на різних пристроях та браузерах, незалежно від того, які шрифти встановлені локально у користувачів.

  Приклад використання веб-шрифтів у CSS:
  @font-face {
    font-family: 'CustomFont';
    src: url('/fonts/CustomFont.woff2') format('woff2'),
        url('/fonts/CustomFont.woff') format('woff');
  }

  body {
    font-family: 'CustomFont', sans-serif;
  }


Google Fonts
  Google Fonts — це безкоштовний сервіс, що надає велику колекцію шрифтів, які можна легко інтегрувати в веб-проекти без необхідності завантаження та вбудовування їх безпосередньо на сервер.

Приклад використання:
<head>
  <link href="https://fonts.googleapis.com/css?family=Roboto&display=swap" rel="stylesheet">
  <style>
    body {
      /* Встановлення 'Roboto' як основного шрифту для тексту на сторінці */
      font-family: 'Roboto', sans-serif;
    }
  </style>
</head>


Font Awesome
  Font Awesome — це бібліотека іконок та інструментів, яка дозволяє нам легко інтегрувати векторні іконки в наші веб-проекти для додавання стилізованих іконок, які можуть покращити інтерфейс користувача та дизайн, не вимагаючи від нас створення власних графічних зображень.
