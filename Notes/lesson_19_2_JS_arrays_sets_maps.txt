Arrays

  General
    Масив (Array) — це структура даних, яка дозволяє зберігати кілька значень в одній змінній.

    В JS масиви також є об’єктами і наслідують властивості глобального об’єкта Object.

    Приклад масива з різними типами даних:
      let arr = [1, "hello", true, {name: "Alice"}];


    Динамічний розмір:
      Можна додавати/видаляти елементи без попереднього визначення довжини

      Приклад використання властивості length
        let colors = ["червоний", "зелений", "синій"];
        console.log(colors.length); // 3

        // Використання в циклі
        for (let i = 0; i < colors.length; i++) {
            console.log(colors[i]);
        }

        // Можна змінити довжину масиву
        colors.length = 2;
        console.log(colors); // ["червоний", "зелений"]


    Створення масиву
      let numbers = [10, 20, 30];   // літеральний синтаксис
      let empty = new Array();      // через конструктор (рідше)


    Отримання елементів масиву по індексу:
      let fruits = ["яблуко", "банан", "груша"];
                      0         1        2
      console.log(fruits[0]);   // "яблуко"
      console.log(fruits[2]);   // "груша"

  Цикли для масивів
    Цикл for для масивів

      Дозволяє пройтись по елементах за індексом.

      for (let i = 0; i < fruits.length; i++) {
        console.log(fruits[i]);
      }


    Цикл for...of

      Простіший спосіб перебору всіх значень масиву.

        let numbers = [10, 20, 30, 40];

        for (let num of numbers) {
            console.log(num);
        }
        // Виведе: 10, 20, 30, 40


    Перевага for…of:
      - Не потрібні індекси
      - Код коротший і читабельніший, особливо коли потрібно перебрати значення елементів

  Основні методи масивів
    https://www.w3schools.com/js/js_array_methods.asp

    Метод	                      Опис	                                    Приклад
    push()	          додає елемент у кінець масиву	                    arr.push("orange")
    pop()	            видаляє останній елемент	                        arr.pop()
    fill(value)	      заповнює масив вказаним значенням	                arr.fill(0)
    at(index)	        повертає елемент за індексом (можна від’ємний)    arr.at(-1)
    indexOf(value)	  повертає індекс елемента або -1	                  arr.indexOf("apple")
    reverse()	        перевертає порядок елементів	                    arr.reverse()
    sort()	          сортує елементи як рядки за замовчуванням	        arr.sort()


Колекції - це структури даних, які дозволяють нам зберігати та керувати даними в організований спосіб. Вони надають різні способи для збереження та маніпуляції з наборами значень або об'єктів.

Map: Колекція пар ключ-значення, де кожен ключ веде до певного значення.

Set: Колекція унікальних значень, де кожне значення може з'являтися лише один раз.

WeakMap: Схожий на Map, але ключами можуть бути лише об'єкти, і він не перешкоджає збору сміття для цих ключів.

WeakSet: Спеціальний тип Set, який містить лише об'єкти та не перешкоджає збору сміття для цих об'єктів.


Sets
  Set — це структура даних, схожа на масив, але:
    - не містить дублікати,
    - не має індексів (елементи зберігаються в порядку додавання).

  Створення Set:
    let set1 = new Set();
    let set2 = new Set(['a', 'b', 'c', 'd']);

  Основні методи Set
    Метод	                Опис	                                    Приклад
    add(value)	    додає елемент	                                set.add(5)
    delete(value)	  видаляє елемент	                              set.delete(2)
    has(value)	    перевіряє наявність елемента	                set.has(3) → true
    clear()	        очищає множину	                              set.clear()
    size	          властивість, що показує кількість елементів	  set.size

  Перебір елементів Set
    Множину можна перебирати циклом for...of або методом forEach():
      let colors = new Set(["red", "green", "blue"]);
      for (let color of colors) {
        console.log(color);
      }

  Перетворення Set у масив
    Іноді потрібно знову отримати масив (наприклад, для сортування):
      let arr = Array.from(colors);
      console.log(arr); // ["red", "green", "blue"]

      або коротше:
      let arr = [...colors];

Maps
  Map — це структура даних, де кожен елемент зберігається у вигляді пари "ключ → значення".

  На відміну від об’єктів:
    - ключем може бути будь-який тип даних (не лише рядок або символ);
    - Mapзберігає порядок додавання елементів.

  Приклад створення:
    let map = new Map();
    map.set("name", "Oleh");
    map.set("age", 25);
    map.set(true, "active");

  Або з масиву пар ключ-значення (рідше):
    const animals = new Map([
    ["cats", 10],
      ["dogs", 20],
      ["parrots", 50]
    ]);

  Основні методи Map
    Метод / Властивість	  Опис	                          Приклад
    set(key, value)	    додає або оновлює елемент	    map.set("city", "Kyiv")
    get(key)	          повертає значення за ключем	  map.get("city")
    has(key)	          перевіряє, чи існує ключ	    map.has("name") → true
    delete(key)	        видаляє елемент	              map.delete("age")
    clear()	            видаляє всі елементи	        map.clear()
    size	              повертає кількість елементів	map.size
    set(key, value)     додає або оновлює значення

  Перебір Map

  Колекцію можна перебирати кількома способами:

    // for…of по парам [ключ, значення]
    for (let [key, value] of map) {
        console.log(key, value);
    }

    // forEach
    map.forEach((value, key) => {
        console.log(key, value);
    });
