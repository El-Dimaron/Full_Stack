Arrays

  General
    Масив (Array) — це структура даних, яка дозволяє зберігати кілька значень в одній змінній.

    В JS масиви також є об’єктами і наслідують властивості глобального об’єкта Object.

    Приклад масива з різними типами даних:
      let arr = [1, "hello", true, {name: "Alice"}];


    Динамічний розмір:
      Можна додавати/видаляти елементи без попереднього визначення довжини

      Приклад використання властивості length
        let colors = ["червоний", "зелений", "синій"];
        console.log(colors.length); // 3

        // Використання в циклі
        for (let i = 0; i < colors.length; i++) {
            console.log(colors[i]);
        }

        // Можна змінити довжину масиву
        colors.length = 2;
        console.log(colors); // ["червоний", "зелений"]


    Створення масиву
      let numbers = [10, 20, 30];   // літеральний синтаксис
      let empty = new Array();      // через конструктор (рідше)


    Отримання елементів масиву по індексу:
      let fruits = ["яблуко", "банан", "груша"];
                      0         1        2
      console.log(fruits[0]);   // "яблуко"
      console.log(fruits[2]);   // "груша"

  Цикли для масивів
    Цикл for для масивів

      Дозволяє пройтись по елементах за індексом.

      for (let i = 0; i < fruits.length; i++) {
        console.log(fruits[i]);
      }


    Цикл for...of

      Простіший спосіб перебору всіх значень масиву.

        let numbers = [10, 20, 30, 40];

        for (let num of numbers) {
            console.log(num);
        }
        // Виведе: 10, 20, 30, 40


    Перевага for…of:
      - Не потрібні індекси
      - Код коротший і читабельніший, особливо коли потрібно перебрати значення елементів

  Основні методи масивів
    https://www.w3schools.com/js/js_array_methods.asp

    Базові методи:
      join - об'єднує елементи масиву в рядок, вставляючи між ними заданий роздільник.
        const elements = ['Вогонь', 'Повітря', 'Вода'];
        console.log(elements.join(' - ')); // "Вогонь - Повітря - Вода"

      reverse - змінює порядок елементів масиву на протилежний.

      sort - сортує елементи масиву.

      concat - об'єднує два або більше масивів або додає окремі елементи до масиву, створюючи новий масив.

      slice - повертає новий масив, що містить копію частини масиву.
        const numbers = [1, 2, 3, 4, 5];
        console.log(numbers.slice(1, 3)); // [2, 3]

      splice - змінює вміст масиву, додаючи, видаляючи або замінюючи елементи, і повертає видалені елементи у вигляді нового масиву.
        const numbers = [1, 2, 4];
        numbers.splice(2, 0, 3); // Додає 3 перед індексом 2
        console.log(numbers); // [1, 2, 3, 4]

      push - додає один або кілька елементів у кінець масиву і повертає нову довжину масиву.
        const animals = ['панда', 'коала'];
        animals.push('броненосець');
        console.log(animals); // ['панда', 'коала', 'броненосець']

      pop - видаляє останній елемент з масиву та повертає його.
        const numbers = [1, 2, 3];
        console.log(numbers.pop()); // 3
        console.log(numbers); // [1, 2]

      shift - видаляє перший елемент з масиву та повертає його.

      unshift - додає один або кілька елементів на початок масиву і повертає нову довжину масиву.

      indexOf - повертає перший індекс, за яким даний елемент можна знайти в масиві, або -1, якщо його немає.
        const animals = ['панда', 'коала', 'броненосець'];
        console.log(animals.indexOf('коала')); // 1

      lastIndexOf - повертає останній індекс, за яким даний елемент можна знайти в масиві, або -1, якщо його немає.
        const numbers = [1, 2, 3, 2, 1];
        console.log(numbers.lastIndexOf(2)); // 3

      includes - визначає, чи містить масив певний елемент, і повертає true або false, з можливістю вказати початковий індекс для пошуку.

      find - повертає перший елемент масиву, що задовольняє задану функцію, або undefined, якщо нічого не знайдено.

    Методи, які викорисовуються зі стрілочними функціями:
      forEach - виконує задану функцію один раз для кожного елемента в масиві.
        const numbers = [1, 2, 3];
        numbers.forEach(number => console.log(number)); // 1 2 3

      map - створює новий масив з результатами виклику заданої функції для кожного елемента масиву.
        const numbers = [1, 2, 3];
        const doubled = numbers.map(number => number * 2);
        console.log(doubled); // [2, 4, 6]

      filter - створює новий масив, включаючи всі елементи, для яких задана функція повертає true.
        const numbers = [1, 2, 3, 4, 5];
        const evens = numbers.filter(number => number % 2 === 0);
        console.log(evens); // [2, 4]

      every - перевіряє, чи всі елементи масиву задовольняють задану умову, і повертає true або false.
        const numbers = [2, 4, 6];
        console.log(numbers.every(number => number % 2 === 0)); // true

      some - перевіряє, чи хоча б один елемент масиву задовольняє задану умову, і повертає true або false.
        const numbers = [1, 2, 3, 4];
        console.log(numbers.some(number => number % 2 === 0)); // true

      reduce - застосовує функцію до акумулятора та кожного значення масиву (зліва направо), зводячи його до одного значення.
        const numbers = [1, 2, 3, 4];
        const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
        console.log(sum); // 10

      reduceRight - застосовує функцію до акумулятора та кожного значення масиву (справа наліво).
        const numbers = [1, 2, 3, 4];
        const sum = numbers.reduceRight((accumulator, currentValue) => accumulator + currentValue, 0);
        console.log(sum); // 10



Колекції - це структури даних, які дозволяють нам зберігати та керувати даними в організований спосіб. Вони надають різні способи для збереження та маніпуляції з наборами значень або об'єктів.

Map: Колекція пар ключ-значення, де кожен ключ веде до певного значення.

Set: Колекція унікальних значень, де кожне значення може з'являтися лише один раз.

WeakMap: Схожий на Map, але ключами можуть бути лише об'єкти, і він не перешкоджає збору сміття для цих ключів.

WeakSet: Спеціальний тип Set, який містить лише об'єкти та не перешкоджає збору сміття для цих об'єктів.


Sets
  Set — це структура даних, схожа на масив, але:
    - не містить дублікати,
    - не має індексів (елементи зберігаються в порядку додавання).

  Створення Set:
    let set1 = new Set();
    let set2 = new Set(['a', 'b', 'c', 'd']);

  Основні методи Set
    Метод	                Опис	                                    Приклад
    add(value)	    додає елемент	                                set.add(5)
    delete(value)	  видаляє елемент	                              set.delete(2)
    has(value)	    перевіряє наявність елемента	                set.has(3) → true
    clear()	        очищає множину	                              set.clear()
    size	          властивість, що показує кількість елементів	  set.size

  Перебір елементів Set
    Множину можна перебирати циклом for...of або методом forEach():
      let colors = new Set(["red", "green", "blue"]);
      for (let color of colors) {
        console.log(color);
      }

  Перетворення Set у масив
    Іноді потрібно знову отримати масив (наприклад, для сортування):
      let arr = Array.from(colors);
      console.log(arr); // ["red", "green", "blue"]

      або коротше:
      let arr = [...colors];

Maps
  Map — це структура даних, де кожен елемент зберігається у вигляді пари "ключ → значення".

  На відміну від об’єктів:
    - ключем може бути будь-який тип даних (не лише рядок або символ);
    - Mapзберігає порядок додавання елементів.

  Приклад створення:
    let map = new Map();
    map.set("name", "Oleh");
    map.set("age", 25);
    map.set(true, "active");

  Або з масиву пар ключ-значення (рідше):
    const animals = new Map([
    ["cats", 10],
      ["dogs", 20],
      ["parrots", 50]
    ]);

  Основні методи Map
    Метод / Властивість	  Опис	                          Приклад
    set(key, value)	    додає або оновлює елемент	    map.set("city", "Kyiv")
    get(key)	          повертає значення за ключем	  map.get("city")
    has(key)	          перевіряє, чи існує ключ	    map.has("name") → true
    delete(key)	        видаляє елемент	              map.delete("age")
    clear()	            видаляє всі елементи	        map.clear()
    size	              повертає кількість елементів	map.size
    set(key, value)     додає або оновлює значення

  Перебір Map

  Колекцію можна перебирати кількома способами:

    // for…of по парам [ключ, значення]
    for (let [key, value] of map) {
        console.log(key, value);
    }

    // forEach
    map.forEach((value, key) => {
        console.log(key, value);
    });
